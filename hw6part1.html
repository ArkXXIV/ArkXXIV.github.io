Welcome to Homework 6 PART 1!




    For Part 1:

      You need to create a web page with a form containing an input box for the user to enter a number with at least 4 decimal positions. The form should also include buttons to submit the data and clear the form, along with a textarea or another element to display the output. Additionally, include a hyperlink to go back to the main page of homework 6. Create a script that validates the input, displaying an error message if the number is invalid. If the input is valid, use various Math functions to perform calculations on the number (rounding, calculating square root, etc.) and present the original number along with the results in the output area. Remember to use "parseFloat()" when reading input and "toFixed()" when presenting the final result.

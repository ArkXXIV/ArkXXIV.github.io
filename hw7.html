Welcome to Homework 7!


    This assignment will have 2 parts:

    • Part 1

      • Create a webpage with a heading, content area, and a form allowing users to dynamically choose styles (background color, underline, bold, italic, font size). Use radio buttons, checkboxes and a dropdown list for style selection. Apply styles instantly without clear or submite buttons. Include a hyperlink for navigation. Write a script to apply and remove selected styles in real-time, handling checkbox unchecking and ensuring cumulative style application. Note the appropriatness of using radion buttons for font size selection due to their mutually exclusive nature.


    • Part 2

      • Create a webpage with a script that saves two pairs of cookies (for the user's name and username) for a page, deleting both pairs when the browser is closed. If it's the user's first visit, present a form to input their name and username; if revisitting in the same browser session, display a "Welcome Back" message with the retrieved name from the saved cookie.
